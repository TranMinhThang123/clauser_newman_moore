# Installation
Step 1: clone this repository
```
git clone https://github.com/TranMinhThang123/clauser_newman_moore.git
```
Step 2: In the CLI run the following command
```
python .\clauset_newmen_moore.py .\KAR.pairs .\output.txt
```
